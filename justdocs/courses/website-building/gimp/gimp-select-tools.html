    GIMP 選取工具快捷指南

    常用選取工具介紹

    工具名稱 功能說明 快捷鍵
    矩形選取 選取方形或矩形區域。按住 Shift 可選取正方形。 R
    橢圓選取 選取圓形或橢圓區域。按住 Shift 可選取正圓。 E
    自由選取 手繪不規則形狀（套索工具），適合快速勾勒輪廓。 F
    魔術棒 (Fuzzy Select) 根據顏色相近度選取連續區域。 U
    顏色選取 選取全圖中所有顏色相近的區域（不需連續）。 Shift + O
    智慧剪刀 自動偵測邊緣進行選取，適合邊界明顯的物體。 I
    Alpha 選取 在圖層面板對圖層按右鍵，選取「從 Alpha 管道取得選取範圍」。 -

    GIMP 進階選取技巧

    1. 前景選取工具 (Foreground Select Tool)

    前景選取工具使用特定的演算法來區分主體與環境, 因為用演算法所以速度慢

    1. 大致選取： 在主體周圍粗略地畫一圈（類似套索工具），然後按 Enter。此時內部會呈現藍色覆蓋。
    2. 塗抹主體： 使用筆刷在主體上塗抹，選取主體內部的代表性顏色。您不需要塗滿整個主體，只要讓 GIMP 辨識出哪些顏色屬於前景即可。
    3. 預覽結果： 再次按 Enter 查看遮罩預覽，GIMP 會自動計算細微的邊緣（如髮絲）。
    4. 完成選取： 確認無誤後，最後按一次 Enter，即可將遮罩轉換為動態虛線選取區。

    2. 快速遮罩 (Quick Mask)

    快速遮罩讓您能以「繪圖」的方式來修改選取範圍。

    • 啟用： 按下 Shift + Q（或點擊影像視窗左下角的小虛線方框圖示）。影像會覆蓋上一層紅色的半透明層。
    • 操作邏輯：
      • 使用白色筆刷： 增加選取範圍（擦除紅色部分）。
      • 使用黑色筆刷： 減少選取範圍（塗上紅色部分）。
      • 使用灰色或低不透明度： 建立羽化（柔化）的選取邊緣。
    • 關閉： 再次按下 Shift + Q，即可將剛才繪製的範圍轉回標準選取區。

    3. 路徑選取 (Paths Tool)

    路徑是數學向量，這意味著它們可以無限調整且邊緣絕對平滑。

    1. 建立路徑： 使用 路徑工具 (B) 在物體邊緣點擊建立節點。點並拖曳可產生貝茲曲線（Bezier curves）。
    2. 封閉路徑： 按住 Ctrl 鍵並點擊起點，即可完成閉合路徑。
    3. 調整細節： 路徑閉合後，您仍可以隨時拖動節點或控制桿來完美貼合輪廓。
    4. 轉換為選取區： 在左側的 工具選項 面板中，點擊 「從路徑建立選取區域」 按鈕，或者使用快捷鍵 Shift + V。
